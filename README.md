# DotNetAstGen

This script creates an Abstract Syntax Tree (AST) of all `.cs` files in JSON format. The AST is created by using the Roslyn .NET compiler.

## Supported languages & dialects

| Language    | Tool used                   | Notes                           |
| ----------- | --------------------------- | ------------------------------- |
| .NET        | Roslyn                      | WIP |
